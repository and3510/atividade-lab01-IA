# Lab P1-01: Implementação do Mecanismo de Self-Attention


## Como Executar o Código

### 1. Pré-requisitos
Certifique-se de ter o [Python 3](https://www.python.org/) instalado em sua máquina.

### 2. Ativando o Ambiente Virtual
O projeto utiliza um ambiente virtual (`venv`) para isolar as dependências. Ative-o utilizando o comando correspondente ao seu sistema operacional:

**No Windows:**
```bash
venv\Scripts\activate

```

**No Linux/macOS:**

```bash
source venv/bin/activate

```

### 3. Instalando as Dependências

Com o ambiente virtual ativado, instale as bibliotecas necessárias (NumPy, Matplotlib e Seaborn) executando:

```bash
pip install -r requirements.txt

```

### 4. Rodando o Script

Execute o arquivo principal para a frase "Meu nome é Anderson e estudo Software" no codigo:

```bash
python script.py

```
